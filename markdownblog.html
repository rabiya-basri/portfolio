Markdown for Beginners

Markdown is a lightweight and easy-to-use syntax for styling all forms of writing on the GitHub platform.

What you will learn:

  • What is Markdown
  • Format README.md using markdown

What is Markdown?

Markdown is way to style text on web. You control the display of the document by formatting text as bold or italic, creating lists, adding images are just a few of the things we can do using markdown. Mostly, we write markdown using these two special characters:

  • #
  • **

Headings

To create heading use # symbole before your heading text.

Ex: # portfolio

The following line in README.MD will be displayed in bold and it will have HTML H1 tag feature addedd to it

portfolio

bold

To make text bold use **text**

Ex: I stay in **Bangalore**

Will be displayed as:

I stay in Bangalore

italic

to make text italic use *text*

Ex: I stay in Bangalore

will be displayed as:

I stay in Bangalore

bold and italic

to make text bold and italic use ***text***

Ex: My country is ***India***

will be displayed as:

MY country is India

lists

to make unordered use -(space)listitem or *(space)listitems

Ex:

  • - itemone
  • - itemtwo
  • - itemthree

will be displayed as:

  • itemone
  • itemtwo
  • itemthree

ordered list

to make ordered use, precede each line with a number.

Ex:

  1. itemone
  2. itemtwo
  3. itemthree

will be displayed as:

  1. itemone
  2. itemtwo
  3. itemthree

Quoting text

you can quote text with a >

Ex: In the words of Abraham Lincoln:

> Pardon my French

Will be displayed as:

| Pardon my French
